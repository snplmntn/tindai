How should Analytics organize the three areas?

Both options keep Overview, Insights, and Predictions & AI. This choice is about navigation feel on a small Android screen.

Option A: Real sub-tabs
Analytics
Daily + monthly sales snapshot

Option A: Real sub-tabs

Each area gets its own focused screen state. Best when the page is feature-rich and users may revisit one area repeatedly.

Option B: Single scroll journey
Analytics
Overview section
Insights section
Predictions & AI section

Option B: Single scroll journey

One page that moves from current state to patterns to predictions. Simpler implementation and easier demo narration, but less separation between sections.

Recommended

  • Option A if Analytics will become a true destination tab.
  • Option B if the goal is the lightest possible MVP.
